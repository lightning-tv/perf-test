# Blits Perf Test

## Getting started

```
git clone https://github.com/chiefcll/blits-perf-test
cd blits-perf-test
pnpm i
pnpm build
pnpm preview
```

Start the chrome dev tools and reload the page and record. Nodes are set to start creating after 2s timeout.
